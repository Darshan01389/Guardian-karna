Guardian Karna Browser – CN Mini Project

How to run (Windows):

1. Install Python 3.12 (or similar) from python.org and check "Add to PATH".
2. Open Command Prompt and run:

   cd path\to\karna
   pip install PyQt6 PyQt6-WebEngine psutil 

3. Start the browser with:

   python main.py

Features:
- Custom secure browser with risk-based scanning.
- Balanced / Moderate / Advanced Defence modes.
- Performance panel (CPU, memory, network).
- Background protection flip view.
- Community side chat.
- Download scanning with Save As dialog.
